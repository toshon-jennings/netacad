# The City School x Cisco NetAcad — Student Tech Dashboard

A single-page student dashboard serving as the central launchpad for The City School's first Cisco NetAcad cohort in Philadelphia, PA.

## What's Inside

- **Cisco NetAcad** — direct link to your core curriculum
- **AI & Developer Tools** — Claude, ChatGPT, Gemini, Jules, Flow, StudioOS
- **Staff Contact** — one-click email to school leadership and coordinators
- **Mobile-responsive** — works on phones, tablets, and desktops

## Deploy to GitHub Pages

1. Push this repo (public).
2. Go to **Settings > Pages**.
3. Source: `main` branch, `/ (root)`.
4. Your dashboard is live at `https://<username>.github.io/<repo>/`.

## Tech Stack

Zero build steps. Pure static HTML + CSS + JS (JS is enhancement-only — all links work without it). Font: Inter via Google Fonts CDN.
